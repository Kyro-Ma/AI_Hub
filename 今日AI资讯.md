## 1. 无问智科重磅发布业界首个物理AI数据基座平台
2026年3月5日无问智科在德清举办了发布会，正式推出业界首个物理AI数据基座平台名为无垠。这个平台能够帮助机器人更好地学习和完成各种任务，比如叠衣服、分拣物品和在商超中导航。它通过采集真实场景数据和生成虚拟场景数据，构建了从数据采集到模型训练的完整流程。平台支持多种场景，包括家庭、商业、物流和工业等，已经积累了上万小时的真实数据和百万级别的虚拟数据。无问智科还与多家企业合作，共同打造具身智能的应用场景。该平台解决了当前机器人学习中数据难获取、质量不高、场景覆盖不全的问题，为机器人走向实际生活提供了坚实基础。此次发布标志着中国在具身智能数据基础设施方面迈出了重要一步。

## 2. 无需排队工业级漫剧制作：360纳米漫剧流水线抢跑AI影视工业化
360集团推出名为纳米漫剧流水线的AI制作平台，能够帮助创作者快速制作出接近电影质量的国风短片。这部爆款作品《霍去病》仅用两天时间完成，成本仅三千元，制作效率远超传统方式。平台通过自研技术将分镜生成成功率提升至九成以上，大幅降低制作难度和成本。单个画面制作费用降至一元五角，一部一百二十集的剧集可在七天内完成，比以往节省了七成时间。平台还推出省钱模式，每分钟制作成本从四十六元七角降到十六元二角。该平台已与多家知名影视公司合作，未来将接入Seedance 2.0引擎，进一步提升制作能力。这一技术为AI漫剧行业提供了高效、低成本的工业化解决方案。

## 3. 企业级OpenClaw最强拍档来了！万亿参数的国产多模态大模型，刚刚开源发布
YuanLab.ai团队近日开源发布源Yuan3.0 Ultra多模态大模型，这是目前少数几个达到万亿参数规模的开源模型之一。该模型采用混合专家结构，通过LAEP方法优化训练过程，使参数减少三成，预训练效率提升近五十个百分点。模型能同时理解图片和文字，擅长处理企业文档、表格和数据库信息，能够准确提取关键数据并完成复杂任务。它在文档理解、信息检索和数据分析方面表现优于CLAUDE、GEMINI等国际主流模型。模型还引入了快速思考机制，避免无意义反复思考，提升回答效率。该模型已全面开放，支持免费下载和使用，为构建企业级智能体提供强大支持。未来还将推出更多版本，助力企业更轻松地应用人工智能技术。

## 4. GitHub Issue 标题的 prompt injection 导致约 4k 开发者机器被攻破
GitHub 一个开源项目的问题导致大约四千名开发者电脑被入侵。攻击者通过在问题标题中写入特殊指令，让一个自动处理工具把这些内容当作命令发送给 CLAUDE 模型。模型没有做好检查就执行了指令，结果在开发者电脑上安装了一个叫 OpenClaw 的恶意程序。这个程序可以悄悄获取电脑权限。攻击利用了 npm 安装时的后处理脚本和 GitHub 允许引用别人仓库的特性，尤其是可以引用别人的分支提交。很多人批评这种自动化工具太危险，因为一旦输入被恶意操控，就可能造成严重后果。社区呼吁平台和开发者加强安全措施，比如限制自动操作权限，要求所有危险操作必须人工确认。人们也担心类似问题会反复出现，因为安全规则没有被真正落实。

## 5. 解放用户注意力：AI 编程助手 Cursor 上线自动化智能体模式
Cursor公司推出了一种新的自动化功能，叫做Automations，帮助程序员减轻管理多个编程助手的负担。过去程序员需要不断手动启动和监控这些助手，现在系统可以自动触发，比如当代码库有新内容、Slack收到消息或者按时间自动运行。这个功能不需要程序员一直操作，而是让助手在合适的时候自动工作，只有在需要时才请人参与。系统每小时能处理数百次任务，不仅能做代码检查，还能应对服务器出问题的情况，比如自动查看日志或者在Slack发送每周的代码更新总结。这种新方式让程序员像在传送带上工作一样，不用时刻盯着，但仍然能及时介入关键环节。

## 6. 免费用：微软 Bing Video Creator 已接入 OpenAI Sora 2 AI 视频生成模型
微软近日宣布其必应视频创作者服务已接入OPENAI的Sora 2视频生成模型。这项升级让生成的视频动作更生动，画面更真实，细节更清晰。用户只需输入文字描述，系统就能自动制作出对应的视频内容。新功能还支持添加音效、人声和背景音乐，让视频听起来更完整。所有必应用户都可以免费使用这项服务，首次可获得十次快速生成机会，之后还能继续使用慢速生成功能。如果用户有积累的奖励积分，也可以用积分换取更多快速生成次数。系统会在每个生成的视频上加上水印，并嵌入符合标准的C2PA凭证，确保视频来源清晰可查。这项更新让普通人也能轻松制作高质量的AI视频。

## 12. 实车体验｜地平线HSD上车iCAR V27：5米硬派方盒子从此不挑路
iCAR V27是一款5米大的硬派SUV，它搭载了地平线HSD智能驾驶系统，能够像人类司机一样灵活应对复杂路况。车辆配备了强大的算力芯片和多种传感器，可以快速处理周围环境信息。系统采用了一种全新的驾驶方式，不需要把驾驶任务分成多个部分，而是直接根据所有信息做出驾驶决策，反应速度比人快很多。它能在狭窄道路、人流车流大的地方平稳通过，不会一直停住。系统还能理解交通标志和复杂场景，比如潮汐车道和施工区域，即使没有导航信号也能自主寻找道路。在高峰时段，它能安全避让行人和车辆，行驶更舒适。这项技术让高阶智能驾驶从高端车走向了大众市场，让更多人能享受到智能驾驶带来的便利。

## 13. Jido 2.0：基于 Elixir/BEAM 的 agent 框架，内置 ReqLLM 与可视化工具
Jido 2.0 是一个基于 Elixir 和 BEAM 的代理框架，它不直接把大模型放进核心，而是把调用大模型的功能单独做成一个工具叫 ReqLLM。这个框架强调数据和纯函数，让代理的运行更稳定可靠。它用 BEAM 的特性可以同时运行成千上万个轻量级代理，适合在小设备上运行，比如树莓派。框架内置了监督机制，能自动处理节点出错、进程崩溃等问题，还能把代理的状态保存到数据库里，方便恢复。作者参考了几十年的代理研究，并用 CLAUDE 等模型测试和优化代码。虽然有人担心实际应用不多，但框架支持与多种工具对接，能帮助用户管理复杂任务。发布后引起社区关注，作者正在开发可视化仪表盘，方便大家查看代理运行情况。

## 14. 亿万富翁科斯拉预言：AI 将胜任 80% 岗位，当今 5 岁儿童成年后无需为生存而工作
